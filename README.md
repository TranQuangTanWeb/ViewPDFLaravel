- Clone project: git clone https://github.com/TranQuangTanWeb/ViewPDFLaravel

- composer install
- Copy .env.example to .env:
       Windows: copy .env.example .env
       Ubuntu: cp .env.example .env
- Run: php artisan key:generate
- Run: php artisan serve
